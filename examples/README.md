## ðŸ“ File Format

All Excel files follow this structure:

| Column | Description |
|--------|-------------|
| Description | Bond identifier/name |
| Settlement Date | Purchase/valuation date |
| Maturity Date | Bond maturity date |
| Coupon Rate | Annual coupon rate (decimal) |
| Face Value | Par value of the bond |
| Redemption | Redemption value at maturity |
| Frequency | Coupon payments per year (1=Annual, 2=Semi-annual, 4=Quarterly) |
| Market Price | Current market price (leave blank to calculate from YTM) |
| YTM | Yield to Maturity (leave blank to calculate from Market Price) |

> **Note**: Either `Market Price` or `YTM` should be provided, not both. The tool will calculate the missing value.
